# Product

## Register

product

## Users

MiniDone is for Mac users who want a small, fast task list for everyday work. They use it while switching between apps, often from the keyboard or menu bar, and need to capture, finish, restore, and organize tasks without ceremony.

## Product Purpose

MiniDone is a native macOS 14+ SwiftUI task manager with a Dock app window and a compact menu bar mini surface. Success means a new user can understand the sidebar, add a task, complete it, and find completed work in under 10 seconds.

## Brand Personality

Quiet, focused, native. The interface should feel like a small system utility, not a marketing page or a heavy project-management suite.

## Anti-references

Avoid oversized typography, decorative gradients, glassy effects, card-heavy layouts, modal-heavy flows, and visual references that make the app feel less native. Do not reintroduce the earlier large dark reference UI as the default interface.

## Design Principles

1. Capture stays obvious: adding a task should be the clearest action on every task surface.
2. Active lists stay clean: completed tasks should move out of the way while remaining recoverable.
3. Native beats novel: prefer standard macOS controls, system typography, and familiar keyboard behavior.
4. Small surfaces stay small: the menu bar version is for quick work only.
5. Language and theme changes should never make the interface feel mixed or broken.

## Accessibility & Inclusion

Aim for WCAG AA contrast, usable keyboard navigation, clear focus states, and meaningful labels for icon-only controls. Motion should be short and state-driven, with no decorative choreography.
